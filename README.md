# Care Playbook

Bazaar Customer Care — Complaint Resolution Playbook

A single-file interactive HTML reference for the Customer Care team covering:

- 🚚 Delivery Issues (Short / Wrong / Extra item)
- ⚠️ Quality Issues (Damaged / Expired / Weight / SKU)
- ⚡ Auto Resolution criteria & decision tree
- 🔄 Escalation Matrix
- 💬 WhatsApp Communication Templates
- 📊 Monitoring & Feedback
- 📖 Knowledge Base v2 (FAQs, Returns, Payments, Credit, Taxation, IRM Matrix)

## Usage

Open `care_playbook.html` in any browser — no internet connection required.

## Live Preview

Enable GitHub Pages on this repo (`Settings → Pages → main branch`) to host it online.
